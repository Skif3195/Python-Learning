1) Перейди в нужную директорию:

cd 'твоя\директория\'

2) Создай локальный репозиторий в выбраной директории:

git init

3) Далее создай связь (Remote) с удаленным репозиторием на GitHub:

-Для этого зайди на GitHub в нужный репозиторий и скопируй его URL
-Далее введи команду:

git remote add origin 'URL\твоего\удаленного\репозитория'

4) Далее тебе нужно убедиться, что в директории, в которой ты находишься, есть необходимые тебе файлы:

git status

-Красным цветом будут выделены файлы незакоммиченые

5) Далее создай коммит:

git commit

6) Далее отправь данные на удаленный репозиторий на GitHub:

git push origin master - отправка коммита на GitHub (origin - имя удаленного, склонированного
репозитория, master - название ветки, в которую мы пушим коммит)
