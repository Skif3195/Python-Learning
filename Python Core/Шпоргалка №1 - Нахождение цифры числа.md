# [Шпоргалка №1 - Нахождение цифры числа]

## `[ num % 10 ]` - Последняя цифра числа

## `[ (num % 10**2) // 10 ]` - Предпоследняя цифра числа.

## `[ (num % 10**n-1) // 10**n-2 ]` - Вторая цифра n-значного числа.

## `[ (num % 10**n) // 10**n-1 ]` - Первая цифра n-значного числа.

## Пример:

`[ num = 1234 ]`
`[ n = 4 ]`

`[ num % 10 == 4 ]`

`[ (num % 10**2) // 10 == 3 ]`

`[ (num % 1000) // 100 == 2 ]`

`[ (num % 10000) // 1000 == 1 ]`
